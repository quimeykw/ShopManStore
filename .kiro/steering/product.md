# ShopManStore

E-commerce platform for online clothing store with admin panel.

## Core Features

- User authentication (login/register) with JWT
- Role-based access (admin/cliente)
- Product catalog with CRUD operations
- Shopping cart functionality
- Multiple payment methods:
  - Credit card processing
  - Mercado Pago integration
  - WhatsApp checkout (5491122549995)
- Admin panel for managing products, users, and system logs

## Default Credentials

- Username: `admin`
- Password: `admin123`

## Language

The application is in Spanish. All UI text, error messages, and user-facing content should be in Spanish.
